kali linux 一般目录结构

只列出相对重要的目录

目录名 目录作用
/bin 存放系统基本程序(ls, cd, cat,等)
/sbin 存放系统程序(fdisk, mkfs, sysctl,等)
/etc 存放配置文件
/tmp 存放临时文件(通常在启动时删除)
/usr/bin 存放应用程序(apt, ncat, nmap,等)
/usr/share 存放应用程序支持和数据文件

一般用法

| 的使用

将前一个命令的输出重定向到另一个命令的输入。

> 和 >> 的使用与区别

> 重定向到新文件, 将前一个命令的输出保存到文件中, 注意如果重定向的文件不存在会自动创建, 如果存在会覆盖原文件中的内容。

>> 重定向到现有文件。用法和前面一样只是不会覆盖原文件内容。

> 重定向 STDERR。
根据规范描述STDIN、STDOUT、STDERR分别定义为: 0、1、2。

kali linux 文字搜索和处理命令

uniq 命令

uniq 是一个用于对文件或标准输入进行去重操作的命令。它会将相邻的重复行合并为一行，只输出不重复的行。如果对一个乱序的文件进行去重，需要先使用sort进行排序。

参数 描述
-c 在输出结果前添加每行的重复次数。
-d 只输出重复的行。
-u 只输出不重复的行。
-f <字段> 忽略指定字段数目的开头字符。
-s <字符数> 跳过指定数目的开头字符。

示例用法：

用法 描述
uniq file.txt 从文件中去除重复行，并输出不重复的行。
uniq -c file.txt 输出每行重复的次数，并在结果前加上计数。
uniq -d file.txt 只输出文件中出现过一次以上的重复行。
uniq -u file.txt 只输出文件中没有重复的行。
uniq -f 2 file.txt 忽略每行开头的前两个字段，然后进行去重操作。
uniq -s 5 file.txt 跳过每行开头的前五个字符，然后进行去重操作。

使用 uniq 命令时，可以根据需要选择适当的参数和示例用法。

sort 命令详解

sort 是一个用于对文本文件进行排序的命令。它按照指定的排序规则对输入进行排序，并将结果输出到标准输出或指定的文件中。

常用参数

参数 描述
-b 忽略行首的空格字符。
-f 忽略大小写进行排序。
-n 按照数值大小进行排序。
-r 逆序排序，以降序输出结果。
-u 去除重复行，只输出唯一的行。
-k <字段> 按照指定字段进行排序。
-t <分隔符> 指定字段的分隔符。
用法 描述
sort file.txt 对文件进行排序，并将结果输出到标准输出。
sort -f file.txt 忽略大小写进行排序。
sort -n file.txt 按照数值大小进行排序。
sort -r file.txt 逆序排序，以降序输出结果。
sort -u file.txt 去除重复行，只输出唯一的行。
sort -k 2 file.txt 按照指定字段进行排序。
sort -t ',' -k 3 file.txt 指定字段的分隔符为逗号，并按照第三个字段排序。

使用 sort 命令时，可以根据需要选择适当的参数和示例用法。希望这个表格能够帮助你更好地理解和记录 sort 命令的各个参数和示例用法！如果有任何问题，请随时提问。

seq 命令

seq 命令用于生成数字或字符的序列。在需要迭代一系列数值的各种脚本场景中非常有用。seq 命令的通用语法如下：

  • 开始值 是序列的起始值。
  • 步长 指定序列中每个数之间的增量，可以是正数或负数。
  • 结束值 是序列的结束值。

以下是 seq 命令的一些常用选项：

  • -s, --separator=字符串：指定数字之间的自定义分隔符。
  • -w, --equal-width：确保所有数字具有相同的宽度，通过在前面填充零实现。
  • -f, --format=格式：使用指定的格式对输出进行格式化，格式可以包含类似 printf 的格式说明符。

kali linux 终端进程管理

前台运行进程会占用终端，对于需要长时间占用的命令可以放在后台处理。在命令结尾添加 & 符号，可将进程添加到后台处理。当然长时间放在后台处理，可能造成进程报错。被挂在后台的进程被称为jobs。

在终端中, ctrl+c是停止当前命令，ctrl+z是暂停当前命令。

后台处理(bg)

bg 命令可将暂停的命令在后台恢复。

jobs 控制(jobs 和 fg)

jobs 命令查看当前终端后台中运行的命令。

fg 是将当前后台运行的命令，拿到前台运行。

JOBID 描述
%Number Number 指的是作业编号，使用 jobs 查看后台进程时[]中的数字即是。
%String 指挂起命令的名称的开头，例如: %ping。
%+OR%% 指当前的 job。
%- 指上一个 job。

进程控制 (ps 和 kill)

ps 命令

ps 查看系统进程最有用的命令之一。列出系统范围内的进程。

常用参数

参数 描述
e 选择所有进程
f 显示完整格式列表(UID, PID, PPID 等)
C 按命令名选择

kill 命令

kill 向进程发送特定的信号。

kill signal <process pid>

signal 默认是 SIGTERM, 为关闭进程的信号。

kali linux 文件和命令监视

文件监视

tail 命令

tail 命令用来命令行查看文件(默认查看最后 10 行)。常用来监视正在写入的日志文件条目。

watch 命令

tail 命令用来定期运行指定的命令，在默认情况下每两秒钟运行一次。

kali linux 命令行文件下载

wget 命令

wget 命令我们将广泛使用它，用于通过 HTTP/HTTPS 和 FTP 协议下载文件。

curl 命令

curl 命令使用多种协议（包括 IMAP/S、POP3/S、SCP、SFTP、SMB/S、SMTP/S、TELNET、TFTP 等）与服务器之间传输数据。渗透测试人员可以使用它来下载或上传文件，并构建复杂的请求。它的最基本用法与 wget 非常相似。

axel 命令

axel 命令通过多个连接从 FTP 或 HTTP 服务器传输文件。这个工具具有广泛的功能。
